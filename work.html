001 : Hungry Again

A random recipe generating React application that leverages TheMealDB API to fetch a hearty meal inspirations. Users can view the ingredients, instructions, and a demonstrated step-by-step video link on youtube.



Tech Stack

  • MERN Stack (MongoDB, Express, React, Node)
  • Redux
  • TheMealDB API
  • Bootstrap/Material-UI
  • Heroku

Source Code

Github Demo


Features

  • Redux action to retrieve random meal leverages axios for a GET command to TheMealDB API which then responds with generally healthy meal recipes.
  • Users can continually generate new recipes.
